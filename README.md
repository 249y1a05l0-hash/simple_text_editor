Simple Text Editor (Python Tkinter)

A simple GUI-based text editor built using Python and Tkinter. This application allows users to create, open, edit, and save text files.

Features
- Create new file
- Open existing text files
- Save files
- Keyboard shortcuts (Ctrl+N, Ctrl+O, Ctrl+S)
- User-friendly interface

Tech Stack
- Python
- Tkinter (GUI)
